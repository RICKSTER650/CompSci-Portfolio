The Mother Board

What is the Motherboard? It is the mainthing containing the principal components. It holds everything in place and it has connectors for circuit to be added. It helps connect everything with each other and without it the computer could not run.

The CPU

The CPU or Central Proccessing Unit is where the computer calculates all the data and processes results. Its the main component that tells what and how the computer should run. Its connected in a socket inside the motherboard.

The RAM

The RAM or the Random Access Memory is the memory of a computer where everything is saved including data,downloads,and other things like that. It is also connected by a socket in the motherboard.

The Hard Drive

The Hard Drive helps the operating system by reading and writing information and knows where all the files and data are. It is also a data storage device that keeps track of everything.

The Mouse, Keyboard, and Monitor

The mouse, keyboard, and monitor are the most basic yet neccessary compontents to a computer. The are what we use to control the computer and create data by clicking and typing things on the computer monitor which lets you see everything you do.
